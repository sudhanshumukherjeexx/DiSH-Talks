## About this repository

This collection brings together **slides, notebooks, datasets, and follow-up resources** from workshops hosted at the **Digital Scholarship Hub (DiSH)** in the [Claire T. Carney Library](https://lib.umassd.edu/). It is maintained for **open reuse**, **student reference**, and **professional documentation** (for example, linking from a CV or portfolio).

If you use or adapt these materials, a short credit back to this repository and DiSH is appreciated.

**Maintainer:** Sudhanshu Mukherjee — [@sudhanshumukherjeexx](https://github.com/sudhanshumukherjeexx) · [LinkedIn](https://www.linkedin.com/in/sudhanshumukherjeexx/).

## Local setup (general)

- **Python workshops:** Use Python 3.10+ and a virtual environment per folder if `requirements.txt` is not present; see each workshop README for imports and optional API keys.
- **Secrets:** Never commit API keys. This repo includes a `.gitignore` rule for `.env` files. Use `.env.example` where provided and keep real keys only on your machine.
